Welcome to Boundless Magento 2 Theme!

Theme Installation

Theme Files Upload

  • Upload via FTP
  • Upload via CPanel
  1. Log into your hosting space via a FTP software. ie, FileZilla
  2. Unzip the file and upload it to the root directory of the project.
  3. Run the upgrade command bin/magento setup:upgrade
  1. Log into your CPanel(Control Panel) by your hosting account or cpanel user information.
  2. Unzip the file and upload it to the root directory of the project.
  3. Run the upgrade command bin/magento setup:upgrade

Local Setup

  1. Unzip Magento Sources:
    • unzip <boundless.zip> -d < magento_root > </magento_root >
  2. Enable Modules and run upgrades:
    • /magento module:enable\
    • harriswebworks/module-core\
    • harriswebworks/module-menumanager\
    • harriswebworks/module-slider\
    • harriswebworks/Productslider\
    • /magento setup:upgrade
    • /magento setup:static-content:deploy
    • /magento cache:flush

Theme Configuration

Boundless provides extremely easy configuration. You can do it from your Mageto Admin panel.


  1. Login to admin panel
  2. Go to Content → Configuration and click on “Edit” option from the action column.

  3. Click on “Applied Theme” drop-down option and select the “Harris Web Works Boundless” theme.
  4. Finally Click on “Save Configuration” button.
  5. Now clean the cache using bin/magento cache:clean

Footer Setup

  1. Login to admin panel
  2. In Admin Panel inside your left Menu-bar you will see the "Menu Manager".
  3. For ABOUT US section:
    Inside Menu Information -> General Information
    1. Enter your menu title (Default: ABOUT US)
    2. Enter a unique identifier name for your footer menu (default: about-us)
    3. Enter your class name to customize the design if needed (Optional)
    4. Click "Save Menu" button
  4. For CUSTOMER SERVICE section:
    Inside Menu Information -> General Information
    1. Enter your menu title (Default: CUSTOMER SERVICE)
    2. Enter a unique identifier name for your footer menu (default: customer-service)
    3. Enter your class name to customize the design if needed (Optional)
    4. Click "Save Menu" button

You can configure the options related to the footer section in Boundless- Setting Panel > Footer

Call your Custom Footer Menu

Calling in XML :

You can call Footer Menu in XML using this code:>

<referenceBlock name="footer-container">
 <container name="hww-footer-area" htmlTag="div" htmlClass="hww-footer-area">
  <container name="hww-footer-container" htmlTag="div" htmlClass="container" after="subscribe">
   <container name="hww-footer-row" htmlTag="div" htmlClass="row">
    <container name="hww-footer-our-company" htmlTag="div" htmlClass="col-md-4 col-sm-4">
     <block class="HarriswebworksMenumanagerBlockMenu" name="footer-about-us" template="Harriswebworks_Menumanager::html/footer-menu.phtml">
     <arguments>
      <argument name="identifier" xsi:type="string">”YOUR IDENTIFIRE NAME”</argument>
     </arguments>
    </block>
   </container>
  </container>
 </container>
</referenceBlock>

Harriswebworks Menu Manager

Create Menu

Before going into steps to create and configure menu items as per our demo, let’s learn about general settings of our Menu and ways it supports to create a menu item.


  1. Login to admin panel
  2. In Admin Panel inside your left Menu-bar you will see the "Menu Manager".

  3. To add new menu click on "Add New Menu"
  4. Inside Menu Information -> General Information -
    1. Enter your menu title (Your choice).
    2. Enter a unique identifier name for your menu (default: hww-menu).
    3. Enter your class name to customize the design if needed (Optional).
    4. Click the “Save Menu" button.
  5. Now clean the cache using bin/magento cache:clean

Create menu Items

To assign necessary menu items (sub-menu) follow the steps below.

  1. Go to the Menu manager and edit the menu you are working on.
  2. Go to “Assigned Menu Items” and click the “Add Item" button.
  3. Enter your menu item title (sub-menu).
  4. You can choose URL Type either Custom URL or, CMS Page or, Category.
    1. For URL Type: Custom URL
      Provide a valid URL in the Custom URL field.
    2. For URL Type: CMS Page
      In the CMS Page field, you can choose the Home page, Enable Cookies, etc. from the drop-down.
    3. For URL Type: Category
      In the Category field, you can choose any of your categories from the drop-down.
  5. After filling up all the fields click “Save”.
  6. Repeat the process as many times you need.
  7. Now clean the cache using bin/magento cache:clean

Calling custom menu

  • Calling via XML
  • Calling via CMS Page/Block
  1. First, create a Menu with the custom identifier (“your-custom-menu-identifier”).
  2. Add all the Custom menu by adding this code in the default.xml file of the intended module like
    app/design/frontend/{your theme vendor name}/{your theme name}/Harriswebworks_Menumanager/layout/default.xml

    3. <referenceBlock name="catalog.topnav" remove="true"/>
    <referenceBlock name="store.menu">
      <block class="Harriswebworks\Menumanager\Block\Menu"  name="custom.navigation"  template="html/menu.phtml"  before="-" ttl="3600">
        <arguments>
          <argument name="identifier" xsi:type="string">  your-custom-menu-identifier     </argument>
        </arguments>
      </block>
    </referenceBlock>
  1. First, create a Menu with the custom identifier (“your-custom-menu-identifier”).
  2. Create a block from the admin panel and add the following code in the content section.

Product-Slider

Featured Product

Before going into steps to create and configure product slider items as per our demo, let’s learn about the general settings of our product slider and the ways it supports creating the product slider.


  1. Go to Store > Settings > Configuration > HARRIS WEB WORKS > Product Slider
  2. To enable the Featured product click on the “Feature Products Enable” drop-down and select yes.
  3. There are more configuration options under the Feature Product Configuration option you can change. If you don't change any attributes value it will run as the default configuration.
    You can access multiple settings like
    1. Endable or disable the section.
    2. Select The number of products shown in the section.
    3. Edit block title.
    4. Configure auto slide.
    5. Show price or not.
    6. Show add to cart or not.
    7. Show review rating or not.
    8. Configure navigation control.
  4. Finally, Click on the “Save Config” button.
  5. Now, Go to Catalogue > Product > Edit product and Select product as a featured product.


  6. It will be like this on the storefront
